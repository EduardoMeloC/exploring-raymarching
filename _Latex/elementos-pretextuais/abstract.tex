This work aims to explore Ray Marching, a lesser-known relative of Ray Tracing that nevertheless remains relevant in modern rendering applications. Its strengths are demonstrated through the implementation of a cloud rendering technique, highlighting the expressive tools provided by the method. To establish a solid foundation, this work examines Signed Distance Functions (SDFs), which enable smooth shape modeling required for organic cloud structures and are difficult to achieve using traditional polygon-based approaches. Additionally, a Ray Marching implementation in GLSL is presented and directly compared with Ray Tracing, emphasizing differences in performance, visual fidelity, and practical use cases.
